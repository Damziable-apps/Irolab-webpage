# Demo Assets

## Extension Interface Image

Place the IROLAB extension interface screenshot here as `extension-interface.png`.

### Required Image
- **Filename**: `extension-interface.png`
- **Recommended size**: 400-500px width for optimal display
- **Format**: PNG with transparency (if applicable)

### What to capture
Take a screenshot of the IROLAB extension popup showing:
- Color Selection section (Pick a Color, Extract Site Colors)
- Test Area with color preview
- Collections view
- Recent colors

The image should have a green border on the left side to match the IROLAB branding.

### Fallback
If the image is not found, a fallback UI will be displayed automatically.
